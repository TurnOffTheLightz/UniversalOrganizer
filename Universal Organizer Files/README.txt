__Universal Organizer__ first open 2019-08-31 at 06:23:34 CEST

__Universal Organizer__ is an app which aims to:
	-help you organize your daily routine or week plan
	-plan and name segments during songwriting process, featuring one-clip records
	-include block-diagram maker feature - for clearer and faster programming process
	-include tempo-detector-on-taps feature
